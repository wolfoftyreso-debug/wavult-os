## 1.1 Positioneringsplattformen

### Kärninsikten

Microsoft säljer inte ett program — de säljer grunden alla program körs på.
pixdrift säljer inte ett verktyg — de säljer grunden hela verksamheten körs på.

Det är en fundamental skillnad. Verktyg byts ut. Operativsystem byts inte ut.

### Analogin

| Windows-ekosystem | pixdrift-ekosystem |
|-------------------|--------------------|
| Windows OS | pixdrift OS |
| Office-appar | pixdrift-moduler (Execution, Capability, Process, Currency, Reports) |
| Tredjepartsappar | pixdrift-integrationer (Slack, Fortnox, HubSpot...) |
| IT-avdelning | pixdrift Admin |
| Aktivitetshanteraren | Execution (vad som rör sig, vem som gör vad) |
| Filhanteraren | Process (hur arbetet struktureras och flödar) |
| Kontrollpanelen | Admin (konfigurera allt, styr åtkomst) |
| Task Scheduler | 22 automatiserade bakgrundsjobb |
| Windows Update | Sömlös SaaS-leverans — alltid uppdaterat, alltid live |

### Tagline-analys och rekommendation

**Nuvarande:** "One System. Every Operation."
— Bra, men generisk. Kommunicerar konsolidering, inte hegemoni.

**Alt 1: "Your Business, Running on pixdrift."**
— Starkast. Sätter pixdrift som substratet, inte som ett verktyg. "Running on" är OS-språk — folk förstår det intuitivt. Det är personligt ("Your Business"), det är aktivt ("Running"), och det namnger produkten i slutet som en naturlig destination. Ingen konkurrent kan använda denna formulering utan att ljuga.

**Alt 2: "The Operating System for Modern Business."**
— Tydligast kategoriskapande. Perfekt för analyst briefings, pressreleaser och B2B-presentationer. Kan upplevas klinisk i direktkommunikation.

**Alt 3: "Run Everything."**
— Kortast och kraftfullast som ren headline. Fungerar utmärkt som sekundär tagline i annonser, men för bred för att vara primär differentiator.

### ✅ REKOMMENDATION

**Primär tagline:** `Your Business, Running on pixdrift.`

**Motivering:** "Running on" är borrowed equity från operativsystemsvärlden — en kognitiv genväg som omedelbart placerar pixdrift som infrastruktur snarare än applikation. Det är vad Intel Inside gjorde för chips. Köparen ska tänka: "Allt vi gör körs på det här." Det är en positionering som inte kan konkurreras bort utan att utmanaren bygger ett lika fullständigt system.

**Sekundär tagline (annonser/rubriker):** `Run Everything.`

**Kategorilabel (B2B-pitch, analyst):** `Business Operating System (BOS)`

## 1.2 Messaging Framework

### Tier 1 — Core Message (används överallt)

> pixdrift är operativsystemet ditt företag körs på.

Enkelt. Orubbligt. Upprepas i varje kanal tills det sitter.

### Tier 2 — Proof Points (används i features och demos)

Varje proof point kopplar pixdrift till ett välkänt OS-koncept:

- **Som Windows har Aktivitetshanteraren** — pixdrift har **Execution**: se allt som rör sig i realtid. Varje affär, varje uppgift, varje kontakt — live.
- **Som Windows har Filhanteraren** — pixdrift har **Process**: strukturera hur arbetet faktiskt flödar, inte hur det råkar flöda.
- **Som Windows har Kontrollpanelen** — pixdrift har **Admin**: konfigurera allt, styr exakt vem som har tillgång till vad.
- **Som Windows har Task Scheduler** — pixdrift har **22 automatiserade bakgrundsjobb**: FX-revalvering, SIE4-generering, compliance-påminnelser — körs utan att någon tänker på det.
- **Som Windows uppdateras sömlöst** — pixdrift är **alltid uppdaterat, alltid live**: inga installationer, inga migrationer, ingen IT-avdelning som krävs.

### Tier 3 — Supporting Evidence (används i säljmöten och due diligence)

- 40 databastabeller, 82+ API-endpoints
- 22 automatiserade bakgrundsjobb
- SIE4, BAS-plan, GDPR-compliance inbyggt
- Multi-valuta: EUR, SEK, GBP, USD + 40 valutor
- ISO 9001-stöd inbyggt i Process-modulen
- 99.9% SLA (Enterprise)
- API-first arkitektur — integreras med allt

## 1.3 Persona-anpassad Messaging

### För CEO / Grundare

> "pixdrift är det första systemet som ger dig faktisk kontroll — inte dashboards som visar historik, utan ett system som styr framåt."

**Längre version:**
Du har hundra verktyg och ändå ingen helhetsbild. pixdrift är operativsystemet som förenar sales, ops, ekonomi och HR i ett — så att du ser hela verksamheten, inte delar av den. Beslutsunderlag på sekunder, inte dagar.

**Pain det adresserar:** Brist på reell kontroll trots tool-överflöd.

---

### För COO / Ops

> "Varje process, varje avvikelse, varje kapacitetsgap — synligt, spårbart, åtgärdbart. pixdrift är hur professionella ops-team arbetar."

**Längre version:**
Operationell excellens kräver att arbetet flödar som det ska — inte som det råkar göra. pixdrift ger dig processer som faktiskt körs, avvikelser som flaggas automatiskt och kapacitetsdata som inte kräver ett Excel-ark för att förstå.

**Pain det adresserar:** Operationell osynlighet, processer som lever i folks huvuden.

---

### För CFO

> "Från ledger till SIE4-export, från multi-valuta till kassaflödesrapport — pixdrift är ekonomisystemet som inte kräver ett separat ekonomisystem."

**Längre version:**
Du behöver inte Visma för att bokföra, Fortnox för att exportera SIE4 och Excel för att förstå valutaexponeringen. pixdrift håller allt — och när revisorn frågar är svaret ett klick bort, inte tre dagars arbete.

**Pain det adresserar:** Fragmenterat ekonomilandskap, manuell rapportering, valuta-chaos.

---

### För IT / Admin

> "RBAC, audit log, GDPR-compliance, API-first. pixdrift är byggt för att integreras, inte isoleras."

**Längre version:**
pixdrift är inte ännu en SaaS-silo som kräver egna kopplingar och undantag i säkerhetspolicyn. Granulär rollkontroll per modul och datanivå. Fullständigt revisionsspår. REST API + webhooks för allt. EU-datalagring. Bygg på det — det håller.

**Pain det adresserar:** Integration-kaos, säkerhetshål, verktyg som inte talar med varandra.

## 1.4 Competitive Reframe

> ⚠️ Endast för intern användning i säljmöten. Publicera aldrig. Juridiskt problem annars.

Hur man snabbt reframer ett konkurrentnamn som dyker upp i kunddialog:

| Konkurrent | Vår respons |
|------------|-------------|
| **Notion** | "Notion är ett anteckningsblock. pixdrift är ett operativsystem." |
| **HubSpot** | "HubSpot hanterar dina kunder. pixdrift hanterar hela verksamheten." |
| **Jira** | "Jira spårar tasks. pixdrift kör verksamheten." |
| **Visma** | "Visma håller räkenskaperna. pixdrift håller ihop allt." |
| **Monday / Asana** | "Det är projekthantering. pixdrift är verksamhetens OS." |
| **Salesforce** | "Salesforce är ett CRM. pixdrift börjar där CRM slutar." |

**Säljregel:** Nämn aldrig konkurrentens namn i en slide. Nämn det bara som svar på direkta frågor — och gör det kort, konfident och utan att hetsas.

## 1.5 Onboarding-narrativ

### Välkomstskärm (ny kunds första login)

> "Välkommen till pixdrift.
> Du har precis installerat operativsystemet för din verksamhet.
> Låt oss sätta upp det."

### Setup-wizard — inspirerad av Windows OOBE

Flödet känns som att sätta upp en ny dator: tydligt, progresivt, aldrig överväldigande.

**Steg 1: Organisation**
"Vad heter ert företag?" → org-setup

**Steg 2: Bransch**
"Vilken bransch?" → välj profil
- SaaS / Tech
- Professionella tjänster
- Handel / E-handel
- Tillverkning / Industri
- VVS / Installation / Hantverk
- Annat

**Steg 3: Team-storlek**
"Hur stort är teamet?"
- 1–10 / 11–50 / 51–200 / 200+

**Steg 4: Moduler**
"Vilka moduler vill du aktivera?"
☑ Execution ☑ Capability ☑ Process ☑ Currency ☑ Reports

**Steg 5: Team**
"Bjud in ditt team" → email-fält, rollval

**Steg 6: Klart**
> "pixdrift är redo. Välkommen till kontrollen."
