pixdrift API Reference

Complete reference for all API endpoints. Base URL: https://api.bc.pixdrift.com


All endpoints require Authorization: Bearer <token>. See Authentication for details. Enter your API key in the top-right corner to use the Try it forms.

AUTH
Bearer Token Authentication
All API requests require an Authorization header with a Bearer token.
curl
Node.js
Python
curl https://api.bc.pixdrift.com/api/contacts \
  -H "Authorization: Bearer YOUR_API_KEY"
const res = await fetch('https://api.bc.pixdrift.com/api/contacts', {
  headers: { 'Authorization': 'Bearer YOUR_API_KEY' }
});
response = requests.get(
  'https://api.bc.pixdrift.com/api/contacts',
  headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

POST
/api/contacts
Create a new contact in your organization.

Request Body

FieldTypeDescription
first_name requiredstringContact's first name
last_name requiredstringContact's last name
emailstringEmail address
phonestringPhone number
company_iduuidAssociated company
titlestringJob title
curl
Node.js
Python
curl -X POST https://api.bc.pixdrift.com/api/contacts \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"first_name":"Anna","last_name":"Svensson","email":"anna@example.com"}'
const response = await fetch('https://api.bc.pixdrift.com/api/contacts', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    first_name: 'Anna',
    last_name: 'Svensson',
    email: 'anna@example.com'
  })
});
response = requests.post(
  'https://api.bc.pixdrift.com/api/contacts',
  headers={'Authorization': 'Bearer YOUR_API_KEY'},
  json={'first_name': 'Anna', 'last_name': 'Svensson'}
)

/api/stripe/create-checkout-session
Create a Stripe checkout session. Returns a redirect URL to Stripe hosted checkout.
curl
Node.js
Python
curl -X POST https://api.bc.pixdrift.com/api/stripe/create-checkout-session \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"plan_id":"growth","success_url":"https://app.pixdrift.com/success","cancel_url":"https://app.pixdrift.com/billing"}'
const { url } = await fetch(
  'https://api.bc.pixdrift.com/api/stripe/create-checkout-session',
  {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      plan_id: 'growth',
      success_url: 'https://app.pixdrift.com/success',
      cancel_url: 'https://app.pixdrift.com/billing'
    })
  }
).then(r => r.json());
window.location.href = url; // Redirect to Stripe
