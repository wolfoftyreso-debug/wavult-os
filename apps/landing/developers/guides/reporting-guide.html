Reports & Finance API Guide

Build custom financial dashboards, automate bookkeeping, and export data to accounting software using the pixdrift Reports and Currency APIs.

Financial reports overview

const BASE = 'https://api.bc.pixdrift.com'
const headers = { 'Authorization': `Bearer ${process.env.PIXDRIFT_API_KEY}` }

// Helper to fetch reports
async function getReport(path, params = {}) {
  const url = new URL(BASE + path)
  Object.entries(params).forEach(([k, v]) => url.searchParams.set(k, v))
  return fetch(url, { headers }).then(r => r.json())
}

// Q1 2026 financial picture
const [pnl, balance, vat, cashflow] = await Promise.all([
  getReport('/api/reports/income-statement', { from: '2026-01-01', to: '2026-03-31' }),
  getReport('/api/reports/balance-sheet', { date: '2026-03-31' }),
  getReport('/api/reports/vat', { from: '2026-01-01', to: '2026-03-31' }),
  getReport('/api/reports/cashflow', { from: '2026-01-01', to: '2026-03-31' })
])

console.log('Revenue:', pnl.revenue)
console.log('Net income:', pnl.net_income)
console.log('Cash balance:', cashflow.closing_balance)
console.log('VAT to pay:', vat.net_vat)

SIE4 export (Swedish bookkeeping)

SIE4 is the standard format for Swedish accounting software (Fortnox, Visma, Björn Lundén). Export your full fiscal year:

# Download SIE4 file
curl "https://api.bc.pixdrift.com/api/reports/sie4?from=2026-01-01&to=2026-12-31" \
  -H "Authorization: Bearer $PIXDRIFT_API_KEY" \
  -o pixdrift_2026.se

# File starts with #FLAGGA 0 etc. — import directly to Fortnox
// Node.js — save to disk
const fs = require('fs')
const sie4 = await fetch(
  `${BASE}/api/reports/sie4?from=2026-01-01&to=2026-12-31`,
  { headers }
).then(r => r.text())

fs.writeFileSync('pixdrift_2026.se', sie4, 'latin1')  // SIE4 uses ISO 8859-1
console.log('✅ SIE4 file saved')

Multi-currency reports

If your organization operates in multiple currencies, pixdrift automatically converts values to your base currency for reports. You can also request conversion:

// Convert deal values to EUR for reporting
const pipeline = await getReport('/api/deals/pipeline/multi', { currency: 'EUR' })

// Multi-currency trial balance in SEK
const trialBalance = await getReport('/api/ledger/trial-balance/multi', { currency: 'SEK' })

// FX exposure (which currencies create risk?)
const fxExposure = await getReport('/api/fx/exposure')
console.log('FX exposure by currency:', fxExposure)

Custom dashboard example

Combine multiple endpoints to build a custom executive dashboard:

async function buildExecutiveDashboard() {
  const [admin, sales, finance, risks, nc] = await Promise.all([
    getReport('/api/dashboards/admin'),
    getReport('/api/dashboards/sales'),
    getReport('/api/dashboards/finance'),
    getReport('/api/risks/matrix'),
    getReport('/api/nc/summary')
  ])

  return {
    summary: {
      users: admin.total_users,
      pipeline_value: sales.pipeline_value,
      revenue_mtd: finance.revenue_mtd,
      cashflow: finance.cashflow,
      high_risks: risks.high?.length ?? 0,
      open_nc: nc.open_count ?? 0
    },
    pipeline: sales.deals_by_stage,
    risk_matrix: risks,
    quality: nc
  }
}

const dashboard = await buildExecutiveDashboard()
console.log(JSON.stringify(dashboard, null, 2))

Automated monthly report

Schedule a cron job to generate and email a monthly P&L report:

// monthly-report.js — run via cron: 0 9 1 * *
const nodemailer = require('nodemailer')

async function sendMonthlyReport() {
  const now = new Date()
  const firstDay = new Date(now.getFullYear(), now.getMonth() - 1, 1)
  const lastDay = new Date(now.getFullYear(), now.getMonth(), 0)

  const from = firstDay.toISOString().split('T')[0]
  const to = lastDay.toISOString().split('T')[0]

  const [pnl, sie4] = await Promise.all([
    getReport('/api/reports/income-statement', { from, to }),
    fetch(`${BASE}/api/reports/sie4?from=${from}&to=${to}`, { headers }).then(r => r.text())
  ])

  const subject = `Monthly Report: ${from} – ${to}`
  const body = `
Revenue: ${pnl.revenue.toLocaleString('sv-SE')} SEK
Expenses: ${pnl.operating_expenses.toLocaleString('sv-SE')} SEK
Net Income: ${pnl.net_income.toLocaleString('sv-SE')} SEK

SIE4 export attached for Fortnox import.
  `

  // Send via nodemailer (configure your SMTP)
  // await mailer.sendMail({ to: 'cfo@company.com', subject, text: body, ... })
  console.log(subject, body)
}
